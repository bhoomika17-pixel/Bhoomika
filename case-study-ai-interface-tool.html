Improving Learnability, Trust, and Workflow Fit in an AI-Assisted Interface Generation Tool

Overview

Led foundational and longitudinal research for an early-stage AI-assisted interface generation tool, helping guide product direction toward workflow-aligned capabilities.

Context

Builder teams sought faster ways to create design-system-aligned interfaces. Leadership explored a generative AI tool to support this process.

Role

Led a multi-phase research effort including concept validation and longitudinal evaluation with cross-functional users.

Research Approach

  • Concept validation with designers and developers
  • Longitudinal evaluation with real workflow usage
  • Follow-up interviews

Key Insights

Learnability gaps slowed adoption

Users struggled to effectively interact with the system and evaluate outputs.

Transparency impacted trust

Unclear system behavior reduced confidence and discouraged iteration.

Workflow integration determined value

Users expected the tool to fit into existing workflows.

Visual iteration was preferred

Users favored iterating on existing designs rather than starting from scratch.

Impact

  • Helped identify misalignment in early feature directions
  • Guided focus toward workflow-aligned capabilities
  • Supported ongoing product direction during development

Reflection

AI tools succeed when they align with user workflows, provide transparency, and support clear mental models.
